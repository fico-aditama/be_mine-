## üöÄ How to Deploy (Vercel)

This project is ready for Vercel.

1.  Install Vercel CLI (optional) or just upload this folder to GitHub.
2.  Login to [Vercel](https://vercel.com).
3.  Import the repo/folder.
4.  Deploy! 

You will get a URL like `https://be-mine-collection.vercel.app`.
